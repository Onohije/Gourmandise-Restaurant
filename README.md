Gourmandise Restaurant Website

*Description
Gourmandise is a sleek and elegant restaurant website designed to showcase our exquisite dining experience. It features a modern photo gallery, Instagram integration, and detailed sections about our offerings.

*Features
Photo Gallery: Scrollable image gallery showcasing the restaurant ambiance and dishes.
Instagram Integration: Interactive icons for social media engagement.
Responsive Design: Optimized for desktop and mobile devices.
Custom Components: Reusable React components for scalability.

*Technologies Used
React
JavaScript (ES6)
CSS
Vite
React Icons
